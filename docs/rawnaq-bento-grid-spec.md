# Rawnaq — ফিচার স্পেক: Bento Grid Layout Widget

*নতুন, স্বতন্ত্র উইজেট — Rawnaq-এর existing মডিউলগুলোর কোনোটাই সরাসরি এক্সটেন্ড করে না, তবে ভিজ্যুয়াল ভাষা (রাউন্ডিং, শ্যাডো, amber accent) বাকিদের সাথে কনসিস্টেন্ট রাখা হবে।*

---

## ১. Overview
Bento Grid হলো Apple-স্টাইল "বেন্টো বক্স"-এর মতো অসম-আকৃতির গ্রিড লেআউট — বিভিন্ন সাইজের কার্ড একসাথে সাজানো, যেখানে কিছু কার্ড বড় (হাইলাইট), কিছু ছোট, কিন্তু পুরো গ্রিডটা একটা পরিষ্কার, সুষম কম্পোজিশন তৈরি করে। এই উইজেট ইউজারকে ড্র্যাগ-ড্রপ বা কন্ট্রোল-ভিত্তিক উপায়ে নিজের বেন্টো গ্রিড বানাতে দেবে — কোড বা CSS Grid জ্ঞান ছাড়াই।

## ২. টার্গেট ইউজার ও ইউজ-কেস
- SaaS/প্রোডাক্ট ল্যান্ডিং পেজ — ফিচার শোকেস সেকশন
- পোর্টফোলিও/এজেন্সি সাইট — সার্ভিস বা কাজের নমুনা প্রদর্শন
- "About Us" পেজ — মিক্সড কন্টেন্ট (ছবি, স্ট্যাট, টেস্টিমোনিয়াল) এক গ্রিডে

## ৩. কোর ফিচার লিস্ট

### ৩.১ Content Controls (Elementor Content Tab)

| কন্ট্রোল | বিবরণ |
|---|---|
| Grid Preset | কয়েকটা প্রি-বিল্ট লেআউট টেমপ্লেট (যেমন: "1 বড় + 4 ছোট", "2x2 সমান", "1 ওয়াইড + 3 স্ট্যাক্ড") — এক ক্লিকে শুরু করার জন্য |
| Grid Columns/Rows | কাস্টম গ্রিড সাইজ সেট করা (যেমন 4x3) |
| Cell Repeater | প্রতিটা সেলের জন্য: Content Type (Image / Icon+Text / Stat Counter / Video / Embed / Custom HTML), Column Span, Row Span |
| Span Control | প্রতিটা কার্ডের জন্য ভিজ্যুয়াল drag-handle দিয়ে কতটা কলাম/রো জুড়ে থাকবে সেট করা (Elementor এডিটরে লাইভ প্রিভিউ সহ) |
| Cell Link | প্রতিটা কার্ড আলাদাভাবে ক্লিকযোগ্য/লিংকড করা যাবে |
| Responsive Rearrange | ট্যাবলেট/মোবাইলে প্রতিটা কার্ডের অর্ডার ও স্প্যান আলাদাভাবে override করা |

### ৩.২ Style Controls
- গ্রিড গ্যাপ (row/column আলাদা করে)
- প্রতিটা সেলের জন্য: background (color/gradient/image), border-radius, padding, shadow, hover effect (lift/zoom/tint)
- Overlay/gradient টেক্সট-রিডেবিলিটির জন্য (ছবির ওপর টেক্সট থাকলে)
- Border/Divider: গ্রিড কোষগুলোর মাঝে হেয়ারলাইন বর্ডার অন/অফ, রঙ
- একটা "Featured" স্টাইল প্রিসেট (amber glow accent বর্ডার) — গুরুত্বপূর্ণ কার্ড হাইলাইট করার জন্য

### ৩.৩ Interaction ফিচার
- হোভারে subtle scale/lift অ্যানিমেশন (CSS-only, jQuery ছাড়া)
- স্ক্রলে প্রতিটা কার্ড staggered fade/slide-in অ্যানিমেশন (IntersectionObserver-ভিত্তিক, Rawnaq-এর অন্য উইজেটে ব্যবহৃত একই প্যাটার্ন)
- ভিডিও/GIF সেলে হোভারে অটো-প্লে, ভিউপোর্ট থেকে বেরোলে পজ (পারফরম্যান্স সচেতন)
- Stat Counter সেল টাইপ হলে ভিউপোর্টে ঢোকার পর কাউন্ট-আপ অ্যানিমেশন

## ৪. টেকনিক্যাল আর্কিটেকচার
- রেন্ডারিং: নেটিভ CSS Grid (`grid-template-columns`/`grid-template-rows` + `grid-column`/`grid-row` span) — কোনো জাভাস্ক্রিপ্ট লেআউট-ইঞ্জিন লাগবে না, ফলে ভীষণ হালকা ও ফাস্ট
- রেসপন্সিভনেস: ব্রেকপয়েন্ট-ভিত্তিক আলাদা grid-template + প্রতিটা সেলের override span — CSS মিডিয়া কোয়েরিতে জেনারেট
- Editor UX: Elementor প্যানেলে drag-resize হ্যান্ডেল (span বাড়ানো/কমানো) — ভিজ্যুয়াল, নাম্বার-ইনপুট না দিয়ে সরাসরি canvas-এ রিসাইজ করা যাবে (Elementor-এর native resize interaction ব্যবহার করে)
- Lazy loading: ছবি/ভিডিও সেলে native `loading="lazy"` + IntersectionObserver দিয়ে অ্যানিমেশন ট্রিগার

## ৫. Gutenberg প্যারিটি
- Gutenberg-এ একটা parent "Bento Grid" ব্লক + InnerBlocks হিসেবে প্রতিটা সেল — প্রতিটা সেল ব্লকের নিজস্ব span কন্ট্রোল (block toolbar/সাইডবারে)

## ৬. Edge Cases / বিবেচনা
- খুব বেশি অসম span (যেমন একটা সেল পুরো গ্রিড জুড়ে) দিলে লেআউট ভেঙে যাওয়া এড়াতে ওয়ার্নিং/অটো-অ্যাডজাস্ট
- ছোট স্ক্রিনে জটিল স্প্যান কনফিগারেশন কীভাবে stack হবে তার জন্য স্মার্ট ডিফল্ট (সব সেল সমান-আকৃতির single-column-এ পড়ে যাওয়া, override না দিলে)
- অনেকগুলো ভিডিও সেল একসাথে থাকলে পারফরম্যান্স — একসাথে একাধিক অটো-প্লে না করা (viewport-এ যেটা centered সেটাই প্লে হবে)
- RTL সাইটে গ্রিড ডিরেকশন flip (CSS `direction` প্রপার্টি সম্মান করে)

## ৭. Priority-এর যুক্তি
এটা সম্পূর্ণ নতুন রেন্ডারিং লজিক (CSS Grid-ভিত্তিক, existing SVG-connector সিস্টেমের সাথে সম্পর্কহীন), তাই ডেভ-এফোর্ট তুলনামূলক বেশি — বিশেষ করে drag-resize এডিটর UX বানাতে। কিন্তু এটা একটা হাই-ভিজিবিলিটি, ট্রেন্ডি ফিচার যেটা মার্কেটিং স্ক্রিনশট/ডেমো সাইটে খুব ভালো দেখাবে এবং নতুন ইউজার আকর্ষণ করার সম্ভাবনা বেশি — তাই ROI ভালো হলেও, স্কোপ ছোট রেখে (প্রথমে ৩-৪টা প্রিসেট + বেসিক span কন্ট্রোল) MVP আকারে শুরু করা ভালো, ফুল ড্র্যাগ-রিসাইজ পরে যোগ করা যেতে পারে।

## ডিজাইন কনসিস্টেন্সি নোট
- Featured কার্ড হাইলাইটে Rawnaq-এর amber glow accent ব্যবহার (Hub Diagram/Flow Chart-এর glow-ray-এর মতো একই রঙ-টোকেন)
